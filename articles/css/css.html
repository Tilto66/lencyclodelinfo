Le CSS, c'est quoi?

Les feuilles de style en cascade, généralement appelées CSS de l'anglais Cascading Style Sheets, forment un langage informatique qui décrit la présentation des documents HTML et XML.
Les standards définissant CSS sont publiés par le World Wide Web Consortium (W3C).
Introduit au milieu des années 1990, CSS devient couramment utilisé dans la conception de sites web et bien pris en charge par les navigateurs web dans les années 2000.

L'id


L'id est comme une carte d'identitée des éléments html.
sur les éléments html, pour les spécifier, on utilise le paramètre id
exemple: <p id="Textes">
Nous pouvons ensuite les utiliser en JavaScript (article très prochainement) ou en CSS.

La classe


La classe est comme un Nom de famille chez les éléments html.
Sur ces derniers, pour les spécifier, on utilise le paramètre class
exemple: <p class="Textes">.
Ces classes servent a selectionner des éléments en masse plutot que de les attribuer la même id pour chacun.

Modifier les éléments d'une classe


Sur cette section, nous allons modifier l'apparence de tous les éléments appartenant a une même classe.
En css, les classes sont préfixées par .
Donc nous allons écrire:
.MaClasse{ }
(Remplacez ma classe par le nom de votre classe.)
Dans les accolades, mettez votre personalization.

Mettre de la couleur

Pour mettre de la couleur, tapez (toujours dans les accolades) color: couleur;Et remplacez couleur par le nom de la couleur EN ANGLAIS !!!!


Example:
.HelloTexte{ color: blue; }

Ce code met le contenu visible des éléments avec la classe HelloTexteen bleu.


Modifier les éléments avec un id spécifique


Sur cette section, nous allons modifier l'appareance de tous les éléments appartenant a une id.
En css, les id sont préfixées par #
